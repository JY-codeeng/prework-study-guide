Open the Console to see what's happening

HTML

  • base element defines the base URL for a webpage
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • use meta name="viewport" content="width=device-width, initial-scale=1.0" for responsive design

CSS

  • A margin indicates how much space we want around the outside of an element
  • Padding indicates how much space we want around the content inside an element
  • * is a universal selector - will apply to everything on page
  • padding shorthand w/two values: first value adds padding to the top and bottom, second value to left and right
  • padding shortand w/four values: top, right, bottom, left

Git

  • git checkout -b branch-name : creates a new branch and switches to it
  • git status : checks what branch we are currently on
  • code .opens VS Code, in the directory you are currently in
  • git add -A: stage changes
  • git commit -m ""
  • git pull origin main updates the local branch
  • git push orign feature/branch namepushes changes to the remote repository

JavaScript

  • A variable is a named container that allows us to store data in our code
  • Control flow is the order in which a computer executes code in a script
  • console.log writes to the developer console
  • Math.floor() and Math.random() are both properties of JavaScript's built in Math object that can be used together to generate a random number
